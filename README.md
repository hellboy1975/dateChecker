# dateChecker

This website is a simple Date Checking Tool written as part of a code challenge.  It uses the following tools:

*   [Silex](http://silex.sensiolabs.org/)
*   [Bootstrap](http://getbootstrap.com/)
*   [Bootstrap 3 Datepicker](http://eonasdan.github.io/bootstrap-datetimepicker/)
*   [JQuery](https://jquery.com/)
*   [PHPUnit](https://phpunit.de)
*   [Composer](https://getcomposer.org/)
*   [Vagrant](https://www.vagrantup.com/)
*   [Git](https://github.com/hellboy1975/date-check)
